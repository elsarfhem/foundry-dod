# CHANGELOG

## 0.1.0

- Initial release of Deck of Destiny system for FoundryVTT
- Implemented core system functionality
  - Basic character sheet
  - Attributes, skills, talents, inventory, traumas and conditions tracking
- Localization support with initial translations for English and Italian
- Compendium packs for cards, macros, and journals
- Basic styling and layout for character sheets and item sheets
- Initial setup for automated build and release process using GitHub Actions
- Documentation for contributing, development setup, and localization
